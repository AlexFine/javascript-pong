Adding collision detection so that the ball rebounds correctly off the paddle makes the game playable for the first time. There is no computer AI, but a 2 player game is now possible.

Using naive collision detection, e.g: 

  ball.x < paddle.x
would work in a lot of cases but can lead to the ball overlapping the paddle in some cases when the frame interval (dt) is large, or the ball 'just misses'.

This is corrected using slightly more complex, but more realistic line intersection calculations.

This also allows for more realistic behavior when the ball 'just misses' and instead bounces off the top or bottom of the bat and out for a goal

The collision detection implemented also detects the movement of the paddle and applies a little spin to change the angle of the ball when appropriate
